# 第7章 I/O设备

**核心概念**

外围设备是主机与用户、其他机器或外部环境之间交换信息的设备。按功能可分输入设备、输出设备、外存设备、数据通信设备、过程控制设备。

磁盘是直接访问设备，磁带是顺序访问设备。磁盘地址通常包括驱动器号、磁头号或盘面号、柱面号或磁道号、扇区号。磁盘常按块与主机交换，常用 DMA。

磁盘指标包括存储密度、容量、平均存取时间、数据传输率。平均存取时间 = 平均寻道时间 + 平均旋转等待时间 + 数据传送时间。

显示设备重点掌握分辨率、颜色数、刷新频率、刷新存储器容量和显存带宽。

**易错辨析**

外设不等于外存，外存只是外设的一类。

平均旋转等待时间不是一整圈，而是半圈。7200RPM 是每分钟 7200 转，即每秒 120 转，一圈约 8.33ms，平均等待约 4.17ms。

颜色数 256 表示每像素 8 位，不是 256 字节。

格式化容量通常小于非格式化容量。

**典型计算或比较题**

磁盘容量 = 磁头数 × 柱面数 × 每道扇区数 × 每扇区字节数。

磁盘平均访问时间 = 平均寻道时间 + 平均旋转等待时间 + 数据传送时间。

数据传输率 = 每秒转数 × 每道容量。

刷新存储器容量 = 水平像素 × 垂直像素 × 每像素位数 / 8。

例：1024×1024 像素，颜色数 256，每像素 8 位，容量 = 1024×1024×8/8 = 1MB。

显存刷新带宽 = 每帧数据量 × 帧频。若只有 50% 带宽用于刷新，总带宽至少乘 2。

**结合官方样题推测的考法**

官方样题可见 7200RPM、平均寻道时间、传输率、512 字节扇区参数，说明磁盘平均访问时间是第 7 章最高优先级小计算。还可能结合 DMA 考高速外设成批传送。

**补充题可能延伸**

10 套整理中有刷新存储器容量题，可能改分辨率、颜色位数、帧频、带宽比例。

RAID 可能作比较：RAID0 速度高但无冗余，RAID1 镜像可靠但容量利用率低，RAID5 分布式奇偶校验。

**最后一周背诵清单**

- 外设分类：输入、输出、外存、通信、过程控制。
- 磁盘直接访问，磁带顺序访问。
- 磁盘时间 = 寻道 + 半圈等待 + 传输。
- 7200RPM 半圈等待约 4.17ms。
- 显存容量 = 像素数 × 每像素位数 / 8。
- 1024×1024、256 色 = 1MB。
