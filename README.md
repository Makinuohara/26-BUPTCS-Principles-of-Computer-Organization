## 复习重点

高优先级内容包括：

- 冯·诺依曼结构、存储程序、机器字长、CPI、MIPS。
- 原码、反码、补码、移码、BCD、IEEE754 单精度浮点数。
- Cache 映射、页式虚拟存储、主存芯片扩展、地址范围划分。
- 指令格式、寻址方式、RISC/CISC、扩展操作码。
- CPU 数据通路、微程序控制、硬布线控制、取指与执行阶段微操作。
- 总线仲裁、总线带宽、同步/异步通信。
- 磁盘平均访问时间、显示存储容量。
- 程序查询、中断、DMA、通道。
